Get more sales offering free shipping!

== Description ==

Offering free shipping will improve your conversion rate.

Show your customers how much more should be spent to get free shipping.
The incentive message is displayed at the cart and checkout pages.

== Installation ==

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer.
1. Activate the plugin.
1. Inform your minimum order value for Free Shipping (you will be warned if WooCommerce Free Shipping is not enabled).
1. Your WooCommerce store now has a powerful way of getting more sales.

== Frequently Asked Questions ==

= What is the plugin license? =

This plugin is released under a GPL license

= Does the minimum value includes taxes? =

No, only product value is used for free shipping calculation.

But there is a paid add-on to include tax values, find it at http://pluginterritory.com/shop/wc-free-shipping-taxes-included/

= Can I exclude some products from free shipping offer? =

No, but there is a paid add-on which does this. Find it at http://pluginterritory.com/shop/wc-free-shipping-selected-products/

= Can I track orders and products that were ordered? =

Surely, just check our pro version Woocommerce Free Shipping at http://pluginterritory.com/shop/woocommerce-free-shipping/
